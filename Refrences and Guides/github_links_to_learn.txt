# Essential Learning Resources for Nuclei Integration

## Official Libraries (Go)
*   **Nuclei Core**: https://github.com/projectdiscovery/nuclei
    *   *Focus*: Look at `v3/pkg/nuclei` for the library usage.
*   **ProjectDiscovery Utils**: https://github.com/projectdiscovery/utils
    *   *Focus*: Helper functions for handling generic PD data structures.

## Dashboard References (Open Source)
*   **Mantra**: https://github.com/MrEmpy/Mantra
    *   *Why*: A good example of a recon/scanning framework (though different stack).
*   **Nuclei-Dashboard (Community)**: Search GitHub for "nuclei dashboard" to see how others structure their SQLite/Python implementations (for contrast).

## Enterprise Integration
*   **DefectDojo**: https://github.com/DefectDojo/django-DefectDojo
    *   *Why*: The industry standard for open-source vulnerability management. Learn their data model.
